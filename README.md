### Instalation
```bash
brew update
brew install tesseract --HEAD
brew install tesseract-lang
pip3 install pytesseract
pip3 install -r requirements.txt
```

Checkout the project and get the converter.py
```bash
python3 converter.py images/27.jpeg
```
